**WP Auto Linker** é um plugin simples para WordPress que substitui automaticamente palavras-chave específicas por links dentro do conteúdo das postagens. É ideal para estratégias de *internal linking* ou gerenciamento de links de afiliados.

## Funcionalidades

- Tipo de post personalizado para gerenciar palavras-chave;
- Meta box para adicionar URLs de destino e configurações de sensibilidade a maiúsculas/minúsculas;
- Substituição automática das palavras-chave por links no conteúdo das postagens;
- Suporte para correspondência sensível ou não a maiúsculas/minúsculas;
- Apenas a **primeira ocorrência** de cada palavra-chave é substituída por post.

O **WP Auto Linker**, automatiza parte do SEO interno do seu site, garantindo uma experiência de navegação mais fluida e estratégica para os visitantes.
